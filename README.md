# From Synapses to Vectors

**Author:** Qiaofeng (Independent Researcher)
**Preprint DOI (Zenodo):** https://doi.org/10.5281/zenodo.18918289
**Preprint DOI (OSF):** https://doi.org/10.17605/OSF.IO/Q6K4E
**Date:** March 2026

## Abstract
This paper develops an interdisciplinary theoretical framework arguing 
that a deep structural correspondence obtains between carbon-based neural 
systems and silicon-based computational systems, and derives the structural 
conditions under which silicon-based systems could give rise to 
self-consciousness.
